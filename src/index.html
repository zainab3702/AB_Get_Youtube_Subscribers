GET YOUTUBE SUBSCRIBERS

  1. Retrive all Subscribers :

    GET http://localhost:3000/subscribers : Response with an array of all subscribers details in the database.

  2. Retrive Subscribers Names and Channels:

    GET http://localhost:3000/subscribers/names : Response with an array of all subscribers with only two feilds, their names and subscribed channel.

  3. Retrive a Subscriber by it's ID:

    GET http://localhost:3000/subscribers/:id : Retrive the details of subscriber with the given ID.
